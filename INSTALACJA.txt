
==================================
  Bleed Tool — Instalacja
==================================

WYMAGANIA
---------
  - Python 3.10 lub nowszy
    Pobierz: https://www.python.org/downloads/
    WAZNE: Podczas instalacji zaznacz "Add Python to PATH"

URUCHOMIENIE
------------

  Windows:
    Dwuklik na plik:  uruchom.bat

  macOS / Linux:
    Dwuklik na plik:  uruchom.command
    (jesli nie dziala: Terminal → cd do folderu → bash uruchom.command)

  Przy pierwszym uruchomieniu program automatycznie zainstaluje
  brakujace biblioteki (PyQt6, PyMuPDF, NumPy, Pillow, cairosvg).
  Wymaga polaczenia z internetem.

  Opcjonalnie: Ghostscript (dla plikow .eps/.ai zapisanych jako PostScript)
    - macOS:  brew install ghostscript
    - Windows: https://ghostscript.com/releases/gsdnld.html
    - Linux:  sudo apt install ghostscript

UZYCIE
------

  GUI:
    1. Przeciagnij pliki PDF/SVG/PNG/JPG do okna programu
       (lub kliknij strefę drop aby wybrac pliki)
    2. Ustaw wielkosc bleed (domyslnie 2.0 mm)
    3. Kliknij "Generuj bleed"
    4. Wynikowe pliki PDF pojawia sie w folderze output/

  CLI:
    python bleed_cli.py plik.pdf
    python bleed_cli.py folder/ -o output/ --bleed 3.0
    python bleed_cli.py --batch ./wszystkie -o ./out --recursive
    python bleed_cli.py --help

  Nazwa pliku wyjsciowego: {stem}_PRINT_{W}x{H}mm_bleed{N}mm.pdf
  Przyklad: logo.pdf -> logo_PRINT_100x50mm_bleed2mm.pdf

ROZWIAZYWANIE PROBLEMOW
------------------------

  1. "Nie znaleziono Python"
     → Zainstaluj Python 3.10+ i upewnij sie, ze jest w PATH

  2. "pip nie dziala" / blad instalacji bibliotek
     → Sprobuj recznie:  python -m pip install -r requirements.txt

  3. GUI nie startuje / brak okna
     → Uruchom z terminala:  python bleed_app.py

  4. Blad SVG: "cairosvg" / "cairo not found"
     → Windows: pip install cairosvg
     → macOS: brew install cairo && pip install cairosvg

  5. Blad EPS: "gs: command not found"
     → Zainstaluj Ghostscript (patrz sekcja WYMAGANIA)

ZAWARTOSC
---------
  bleed_app.py        — Interfejs graficzny (PyQt6, glowny program)
  bleed_cli.py        — Interfejs linii polecen (single + --batch)
  config.py           — Konfiguracja (DEFAULT_BLEED, SPOT_COLORS, PLOTTERS)
  models.py           — Struktury danych (Sticker, Placement, Sheet)
  gui/                — Moduly GUI: main_window, workers, widgets, theme
  modules/            — Pipeline: contour, bleed, export, nesting, panelize,
                        marks, crop, preflight, svg_convert, pdf_metadata,
                        ghostscript_bridge
  profiles/           — ICC profile (opcjonalnie CoatedFOGRA39.icc)
  uruchom.bat         — Launcher Windows
  uruchom.command     — Launcher macOS/Linux
  requirements.txt    — Lista zaleznosci Python
